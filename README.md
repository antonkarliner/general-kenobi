# general-kenobi

A Claude skill that responds to "Hello there" with "General Kenobi."

That's it. That's the skill.

## Installation

**Claude Code:**
```
/plugin marketplace add antonkarliner/general-kenobi
```

**Manual:**
```bash
cp -r general-kenobi ~/.claude/skills/
```

## Usage

Say `Hello there` to Claude. Receive `General Kenobi.`
